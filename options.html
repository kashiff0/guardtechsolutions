Your API key is stored locally in Chrome and never sent anywhere except directly to api.anthropic.com

How to Get Your API Key

  • Go to console.anthropic.com and sign in
  • Navigate to API Keys in the left sidebar
  • Click "Create Key" and copy the key (it starts with sk-ant-)
  • Paste it above and click Save Key
  • The extension uses the claude-opus-4-8 model for best message quality

Privacy & Security

  • Your API key is stored locally using Chrome's encrypted storage — it never leaves your browser except in direct API calls to Anthropic
  • Profile data scraped from LinkedIn is sent to the Claude API to generate messages, but is not stored by this extension
  • GuardTech Solutions does not collect any data from this extension
